# Senior Thesis: AAPL vs MSFT Comparative Analysis

## Introduction
This repository contains the research and analysis conducted for my senior thesis, which compares Apple Inc. (AAPL) and Microsoft Corporation (MSFT) based on several financial metrics and market position.

## Features
- Detailed financial analysis of AAPL and MSFT
- Data visualization of stock performance over time
- Comparative metrics and insights

## Technology Stack
- Python for data analysis
- Matplotlib and Seaborn for data visualization
- Pandas for data manipulation

## Usage
1. Clone the repository.
2. Install the required packages using `pip install -r requirements.txt`.
3. Run the analysis script to generate insights.
